# Getting started

### Requirements

* rails 7.0.2.2
* ruby 3.0.0
* postgresql – `brew install postgresql`,
* yarn – `brew install yarn`
* redis – `brew install redis`
* foreman – `gem install foreman`

### Installation

`bin/rails bundle install --no-upgrade`

`yarn install`

`bin/rails db:create db:migrate`

`bin/dev`
